# Outil de suivi du temps de travail

Application web locale de suivi du temps de travail.

## Objectif
Créer une V1 locale permettant :
- une tâche active
- un suivi du jour
- un suivi hebdomadaire
- un suivi mensuel
- une vue d’ensemble
- un historique simple
- un stockage local robuste

## Stack prévue
- React
- TypeScript
- Vite
- Tailwind CSS
- Dexie / IndexedDB

## Statut
Phase de cadrage et préparation technique.
